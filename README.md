## 🛠️ Installation & Setup

1. *Clone the repository:*
   bash
   git clone https://github.com
   

2. *Navigate to the project directory:*
   bash
   cd rock_paper_scissors
   

3. *Open the game:*
   * *For Web/Browser-based:* Open the index.html file directly in your browser.
   * *For Node.js console-based:* Run the command below:
   bash
   node index.js
